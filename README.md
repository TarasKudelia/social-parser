### Requirements:
- python
- pip packages
- playwright

Install python packages:\
`pip install requirements.txt`

Install NPM.

Install playwright packages:\
`playwright install`

---
### Run parser:

`$ python main.py
    -yt [youtube_list.txt] 
    -ig [instagram_list.txt]
    -d [dd.mm.yyyy]` \
All flags (-yt, -ig, ...) are optional and you can specify as many as you want.s
